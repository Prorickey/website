FTC Metrics is a scouting and analytics platform for FIRST Tech Challenge teams. It enables match data collection through mobile-first forms with offline capability, computes EPA and OPR rankings using official FTC Events API data, and predicts match outcomes. The platform features real-time updates via WebSockets, team management with role-based access, and a progressive web app design. I built this as a monorepo with separate web, API, database, and shared packages.
